# AI Campus Assistant – Frontend

This repository contains the **frontend mobile application** for the AI Campus Assistant project.  
The application is built using **Flutter (Dart)** and provides a chat-based interface that allows users to interact with the AI-powered campus assistant.

The app communicates with a backend API that processes queries and returns AI-generated responses based on university data.

-----------------------------------------------------------------------------------------------------------------------------------

## Platform Support
Currently, this project is **configured for Android only**.

iOS, Web, or Desktop support has **not been configured yet** and may be added in future versions.

-----------------------------------------------------------------------------------------------------------------------------------

## Features
- AI-powered chatbot interface
- Clean and simple user interface
- User authentication screens
- Query and response interaction
- Feedback option for incorrect answers
- Integration with backend API
- Multiple GPA calculators for different purposes (e.g., semester GPA, cumulative GPA, and other academic calculations)

-----------------------------------------------------------------------------------------------------------------------------------

## Tech Stack
- Flutter
- Dart
- REST API Integration
- Material UI Components

-----------------------------------------------------------------------------------------------------------------------------------

## Important Notes
- The project is currently **Android-only**.
- The file `theme.dart` exists in the project but **is not being used** due to some configuration issues.  
  The app currently uses default styling and inline theme settings.
- Future updates will include proper theme integration.

-----------------------------------------------------------------------------------------------------------------------------------

## Project Context
This frontend is part of the AI Campus Assistant, a mobile application designed to help students and visitors get university-related information through an AI chatbot powered by a Retrieval-Augmented Generation (RAG) system.
