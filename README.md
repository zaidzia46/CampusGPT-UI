# AI Campus Assistant – Frontend

This repository contains the **frontend mobile application** for the **AI Campus Assistant** project.

The application is built using **Flutter (Dart)** and provides a chat-based interface that allows users to interact with the AI-powered campus assistant.

The app communicates with a backend API that processes user queries and returns AI-generated responses based on university data and documents through a Retrieval-Augmented Generation (RAG) system.

---

## Platform Support

Currently, this project is **configured for Android only**.

Support for **iOS, Web, and Desktop platforms** has not been configured yet and may be added in future releases.

---

## Features

* AI-powered chatbot interface
* Clean and user-friendly UI
* User authentication screens
* Query and response interaction
* Feedback system for incorrect or incomplete answers
* Integration with backend REST APIs
* Multiple GPA calculators for different academic purposes:

  * Semester GPA Calculator
  * Cumulative GPA Calculator
  * Additional academic calculation tools
* Chat history and interactive conversations

---

## Tech Stack

* **Flutter**
* **Dart**
* **REST API Integration**
* **Material Design Components**
* **GetX**

---

## Project Context

This frontend application is part of the **AI Campus Assistant**, a mobile application designed to help students, faculty members, and visitors obtain university-related information through an AI-powered chatbot.

The system utilizes a **Retrieval-Augmented Generation (RAG)** architecture to provide accurate, context-aware, and university-specific responses based on official documents and knowledge sources.
